# BotSystem

BotSystem is a proxy for your bot project, to simplify your code and to get a unique customizable bot for differents platforms.

BotSystem give you the possibility to build a bot which interact with several platform in the same time. (Work In Progress)
BotSystem can be use to build a bot for a unique platform, and in the futur change the platform, without worry about the differents platforms particularities.

Platform available for now:
- Discord

It gives you an environment to build complex bot with a component architecture.
Each component can represent a feature for your bot, which can be connected each other.
Each component can have a configuration file, to custom it as your wish, for your diffrents project.
BotSystem have a configuration file, to choose the components you want for your project.


BotSystem is the prefect tool for:
- complex bot, needed component architecture
- multi platform bot (work in progress)
- customizable bot for private group

## BotSystem story

BotSystem is born in a private Discord friend server, with the idea to build a bot for animation and games, with our own private jokes. Step after step, the bot is becoming bigger with many ideas.
To enable us to share our work, get a better architecture, and have the possibility the change platform, we built BotSystem:
- Component architecture bot, to build complex bot
- Customizable bot, so that everyone can make it their own
- Multi-platform bot, to have the possibility to change platform
